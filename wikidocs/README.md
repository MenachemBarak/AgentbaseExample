# Wikidocs

The **wikidocs** folder is DriveEasy's knowledge base — the durable, human-readable
playbooks and reference docs agents consult and improve over time. It is split into
`internal/` (staff-only) and `public/` (customer-facing). The `SelfEvolve` principle
feeds lessons from failed sessions back into these pages.

## internal/
- **ops-procedures** — pickup, return, and inspection checklists for the Rental
  Counter and Returns & Inspection subteams.
- **pricing-playbook** — how the pricing engine sets rates by vehicle class, demand,
  and season; override rules.
- **fleet-rebalancing** — Sofia Marin's guidance for moving cars between airport
  branches.
- **escalation-runbook** — who to page, mirroring the `principales/` Escalation chain.

## public/
- **customer-faq** — booking, changes, fuel, and mileage questions.
- **rental-policies** — age, license, insurance, late-return, and cancellation terms.

Agents cite the relevant wikidoc when they act, so a quote or a policy answer always
traces back to a versioned source.
