# Role — Backend Engineer (role_backend-engineer)

## Definition
The **Backend Engineer** owns the services and data behind the DriveEasy platform.
This role's exemplar holder is **Raj Mehta**. He works on the back end of
`booking-system` (reservations, accounts, the rental lifecycle) and the
`pricing-engine` (rate calculation), reporting to Tech Lead **Omar Haddad**.

## Responsibilities
- Build and fix the back-end services, APIs, and data behind bookings and pricing.
- Reproduce server-side bugs, trace root cause across services, and fix them.
- In the bug-solving squad, act as the **backend engineer** agent — owning any fix
  in the service or data layer.
- Keep the rental lifecycle (reserve → pickup → in-use → return → invoice) correct
  end to end on the server.

## Purpose
Raj exists so the logic and data behind every rental are correct — turning
back-end incidents into verified fixes that keep bookings and pricing trustworthy.
