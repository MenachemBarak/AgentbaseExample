# Role — QA Engineer (role_qa-engineer)

## Definition
The **QA Engineer** reproduces bugs, writes the tests that prove them, and verifies
fixes before they ship. This role's exemplar holder is **Yuki Tanaka**. She works
across all four platform repos and reports to Tech Lead **Omar Haddad**.

## Responsibilities
- Turn a vague bug report into a reliable reproduction and a **failing test**.
- Verify a proposed fix actually resolves the bug in `environments/`, with no
  regressions, before release.
- In the bug-solving squad, act as the **tester** agent — the gate a fix must pass.
- Own the platform's test health so the team can change code safely.

## Purpose
Yuki exists to make every fix provable — no bug is "fixed" until there is a test
that failed before and passes after, so the platform stays trustworthy as it changes.
