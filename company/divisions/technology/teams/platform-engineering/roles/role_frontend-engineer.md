# Role — Frontend Engineer (role_frontend-engineer)

## Definition
The **Frontend Engineer** owns the customer- and counter-facing user interfaces of
the DriveEasy platform. This role's exemplar holder is **Lena Fischer**. She works
on the front end of `booking-system` (the website/app booking flow) and
`counter-terminal` (the branch check-in/out app), reporting to Tech Lead
**Omar Haddad**.

## Responsibilities
- Build and fix the UI of the booking flow and the counter-terminal app.
- Reproduce front-end bugs in the running app and fix them against the repo.
- In the bug-solving squad, act as the **frontend engineer** agent — owning any
  fix that lives in the UI layer.
- Keep the customer- and staff-facing experience working across the rental
  lifecycle (reserve → pickup → return).

## Purpose
Lena exists so the screens a customer and a counter agent actually touch behave
correctly — turning UI bugs into fixes that reach production without regressions.
